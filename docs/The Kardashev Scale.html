မိတ်ဆွေ... ညဖက် ကောင်းကင်ယံကို မော့ကြည့်မိတဲ့အခါ "ဒီစကြဝဠာကြီးထဲမှာ ငါတို့ကလွဲပြီး တခြားအသိဉာဏ်မြင့်မားတဲ့ သက်ရှိတွေ ရှိနေဦးမလား" လို့ တွေးမိဖူးပါသလား။ ဒီမေးခွန်းဟာ လူသားထုတစ်ရပ်လုံးကို ရာစုနှစ်ပေါင်းများစွာ ခြောက်လှန့်နေတဲ့ မေးခွန်းတစ်ခုဖြစ်ပါတယ်။

စကြဝဠာ ယဉ်ကျေးမှုများ၏ မှတ်ကျောက် သို့မဟုတ် Kardashev Scale

မိတ်ဆွေ... ညဖက် ကောင်းကင်ယံကို မော့ကြည့်မိတဲ့အခါ "ဒီစကြဝဠာကြီးထဲမှာ ငါတို့ကလွဲပြီး တခြားအသိဉာဏ်မြင့်မားတဲ့ သက်ရှိတွေ ရှိနေဦးမလား" လို့ တွေးမိဖူးပါသလား။ ဒီမေးခွန်းဟာ လူသားထုတစ်ရပ်လုံးကို ရာစုနှစ်ပေါင်းများစွာ ခြောက်လှန့်နေတဲ့ မေးခွန်းတစ်ခုဖြစ်ပါတယ်။ ဒီမေးခွန်းကို သိပ္ပံနည်းကျ ချဉ်းကပ်ဖို့အတွက် ၁၉၆၄ ခုနှစ်မှာ ရုရှားနက္ခတ္တဗေဒပညာရှင် Nikolai Kardashev က စိတ်ဝင်စားစရာကောင်းတဲ့ အတိုင်းအတာတစ်ခုကို ဖန်တီးခဲ့ပါတယ်။ အဲဒါကတော့ ယနေ့ ကျွန်တော် ဆွေးနွေးမယ့် "Kardashev Scale" ပဲ ဖြစ်ပါတယ်။

ကျွန်တော်တို့ လူသားတွေအနေနဲ့ နိုင်ငံတစ်ခု၊ လူ့အဖွဲ့အစည်းတစ်ခုရဲ့ တိုးတက်မှုကို တိုင်းတာတဲ့အခါ စီးပွားရေး (GDP)၊ နည်းပညာ စတာတွေနဲ့ တိုင်းတာလေ့ရှိပါတယ်။ ဒါပေမဲ့ Kardashev ရဲ့ အမြင်ကတော့ ပိုမိုကျယ်ပြန့်ပြီး နက်ရှိုင်းပါတယ်။ သူက ယဉ်ကျေးမှုတစ်ခု (Civilization) ရဲ့ အဆင့်အတန်းကို "စွမ်းအင်အသုံးပြုနိုင်စွမ်း" (Energy Consumption) ပေါ်မူတည်ပြီး အဓိက အဆင့် (၃) ဆင့် ခွဲခြားသတ်မှတ်ခဲ့ပါတယ်။

Type I: The Planetary Civilization (ဂြိုဟ်လုံးဆိုင်ရာ ယဉ်ကျေးမှု)

ပထမဆုံးအဆင့်ကတော့ Type I ပါ။ ဒီအဆင့်ကိုရောက်တဲ့ ယဉ်ကျေးမှုတစ်ခုဟာ မိမိတို့နေထိုင်ရာ ဂြိုဟ် (ဥပမာ - ကမ္ဘာဂြိုဟ်) ပေါ်မှာ ကျရောက်လာသမျှ နေရောင်ခြည်စွမ်းအင် အပါအဝင် ဂြိုဟ်ရဲ့ စွမ်းအင်အရင်းအမြစ်အားလုံးကို ပြီးပြည့်စုံစွာ ထိန်းချုပ်အသုံးချနိုင်စွမ်း ရှိပါတယ်။ ဒီအဆင့်မှာဆိုရင် မိတ်ဆွေ စဉ်းစားကြည့်ပါ။ ရာသီဥတုကို စိတ်ကြိုက်ထိန်းချုပ်နိုင်မယ်၊ မုန်တိုင်းတွေကို တားဆီးနိုင်မယ်၊ မြေငလျင်တွေကို ကြိုတင်စီမံခန့်ခွဲနိုင်ပါလိမ့်မယ်။ ကျွန်တော်တို့ လက်ရှိလူသားတွေဟာ ဒီအဆင့်ကို ရောက်ပြီလားလို့ မိတ်ဆွေ မေးစရာရှိပါတယ်။ အဖြေကတော့ "မရောက်သေးပါ"။ ရူပဗေဒပညာရှင် Michio Kaku ရဲ့ တွက်ချက်မှုအရ ကျွန်တော်တို့ဟာ လက်ရှိမှာ Type 0.72 - 0.73 ဝန်းကျင်မှာသာ ရှိပါသေးတယ်။ ကျွန်တော်တို့ဟာ Type I အဆင့်ကိုရောက်ဖို့ နောက်ထပ် နှစ်ပေါင်း ၁၀၀ မှ ၂၀၀ ကြား ကြိုးစားရဦးမှာ ဖြစ်ပါတယ်။

Type II: The Stellar Civilization (ကြယ်စနစ်ဆိုင်ရာ ယဉ်ကျေးမှု)

Type II ကတော့ ပိုပြီး စိတ်လှုပ်ရှားဖို့ ကောင်းပါတယ်။ ဒီအဆင့်ရောက်ရင် ဂြိုဟ်တစ်ခုတည်းရဲ့ စွမ်းအင်နဲ့ မလုံလောက်တော့ပါဘူး။ မိမိတို့ရဲ့ မိခင်ကြယ် (ဥပမာ - နေမင်းကြီး) က ထုတ်လွှတ်သမျှ စွမ်းအင်အားလုံးကို ရယူအသုံးချနိုင်တဲ့ အဆင့်ဖြစ်ပါတယ်။ သိပ္ပံပညာရှင်တွေက ဒီအဆင့်အတွက် "Dyson Sphere" လို့ခေါ်တဲ့ သီအိုရီတစ်ခုကို တင်ပြထားပါတယ်။ ဒါကတော့ ကြယ်တစ်စင်းလုံးကို ဧရာမ ဆိုလာပြားတွေ ဒါမှမဟုတ် စက်ပစ္စည်းတွေနဲ့ ဝန်းရံပြီး စွမ်းအင်ထုတ်ယူမယ့် နည်းပညာဖြစ်ပါတယ်။ ဒီအဆင့်ကို ရောက်ရှိတဲ့ ယဉ်ကျေးမှုတစ်ခုဟာ မျိုးသုဉ်းပျောက်ကွယ်သွားနိုင်ခြေ အလွန်နည်းပါးသွားပါပြီ။ ဘာကြောင့်လဲဆိုတော့ သူတို့ဟာ အာကာသထဲက တခြားဂြိုဟ်တွေကိုပါ လွယ်ကူစွာ ကိုလိုနီပြုနိုင်ပြီး ကြယ်တစ်စင်း ပေါက်ကွဲရင်တောင် ရှင်သန်နိုင်မယ့် နည်းပညာကို ပိုင်ဆိုင်ထားလို့ပဲ ဖြစ်ပါတယ်။

Type III: The Galactic Civilization (ဂလက်ဆီဆိုင်ရာ ယဉ်ကျေးမှု)

နောက်ဆုံးအဆင့်ဖြစ်တဲ့ Type III ကတော့ ကျွန်တော်တို့ရဲ့ ဉာဏ်မှီသမျှ ထိပ်ဆုံးအဆင့်လို့ ဆိုနိုင်ပါတယ်။ ဒီယဉ်ကျေးမှုဟာ မိမိတို့နေထိုင်ရာ ဂလက်ဆီ (ဥပမာ - Milky Way) တစ်ခုလုံးမှာရှိတဲ့ ကြယ်ဘီလီယံပေါင်းများစွာရဲ့ စွမ်းအင်ကို ထိန်းချုပ်နိုင်စွမ်း ရှိပါတယ်။ ဒီအဆင့်မှာ သူတို့ဟာ စကြဝဠာကို ဖြတ်သန်းသွားလာဖို့အတွက် တွင်းနက် (Black Holes) တွေကိုတောင် စွမ်းအင်အရင်းအမြစ်အဖြစ် အသုံးပြုကောင်း အသုံးပြုနိုင်ပါတယ်။ ကျွန်တော်တို့ လူသားတွေရဲ့ အမြင်နဲ့ ကြည့်မယ်ဆိုရင် ဒီအဆင့်ရောက်နေတဲ့ သက်ရှိတွေဟာ "ဘုရားသခင်" တမျှ တန်ခိုးကြီးမားသူတွေအဖြစ် မြင်တွေ့ရမှာ ဖြစ်ပါတယ်။

သုံးသပ်ချက်

Kardashev Scale ဟာ ကျွန်တော်တို့ကို နှိမ့်ချတတ်စေတဲ့ သင်ခန်းစာတစ်ခု ပေးပါတယ်။ မိတ်ဆွေနဲ့ ကျွန်တော်တို့ နေထိုင်တဲ့ ကမ္ဘာကြီးဟာ ဘယ်လောက်ပဲ တိုးတက်နေပါစေ၊ စကြဝဠာကြီးရဲ့ အတိုင်းအတာနဲ့ ယှဉ်လိုက်ရင် ကလေးကစားစရာ အဆင့်မှာပဲ ရှိနေပါသေးတယ်။ ဒါပေမဲ့ စိတ်မပျက်ပါနဲ့ မိတ်ဆွေ။ ဒီ Scale ဟာ ကျွန်တော်တို့ လူသားတွေ လျှောက်လှမ်းရမယ့် လမ်းပြမြေပုံ (Roadmap) တစ်ခုလည်း ဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ ရုပ်ကြွင်းလောင်စာတွေကို မှီခိုနေရတဲ့ အဆင့်ကနေ သန့်ရှင်းသော စွမ်းအင်တွေကို အပြည့်အဝ အသုံးချနိုင်တဲ့ အဆင့်ကို ကူးပြောင်းဖို့ ကြိုးစားရပါလိမ့်မယ်။ အနာဂတ်ဟာ စိန်ခေါ်မှုတွေ များပြားပေမဲ့၊ ဖြစ်နိုင်ချေတွေကလည်း အကန့်အသတ်မရှိဘူးဆိုတာကို ဒီသီအိုရီက မီးမောင်းထိုးပြနေခြင်းပဲ ဖြစ်ပါတယ်။

Kardashev Scale သည် ၁၉၆၄ ခုနှစ်တွင် ရုရှားနက္ခတ္တဗေဒပညာရှင် Nikolai Kardashev ပြုစုခဲ့သော "Transmission of Information by Extraterrestrial Civilizations" အမည်ရှိ သုတေသနစာတမ်းတွင် တရားဝင် စတင်ပါရှိခဲ့ခြင်း ဖြစ်ပါသည်။ ဤသီအိုရီကို နိုင်ငံတကာအဖွဲ့အစည်းတစ်ခုခုက တရားဝင်ဥပဒေအဖြစ် ပြဋ္ဌာန်းထားခြင်းမျိုး မဟုတ်သော်လည်း၊ SETI (Search for Extraterrestrial Intelligence) ကဲ့သို့သော ကမ္ဘာ့ထိပ်တန်း သုတေသနအဖွဲ့အစည်းများနှင့် Michio Kaku ၊ Carl Sagan ကဲ့သို့သော ကမ္ဘာကျော် ရူပဗေဒပညာရှင်များက စကြဝဠာဆိုင်ရာ လေ့လာမှုများတွင် စံသတ်မှတ်ချက် မူဘောင် (Framework) တစ်ခုအဖြစ် ကျယ်ကျယ်ပြန့်ပြန့် လက်ခံအသုံးပြုလျက် ရှိပါသည်။
