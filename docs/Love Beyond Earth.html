လကမ္ဘာထက်မှာ ကျန်ရစ်ခဲ့တဲ့ ဖခင်မေတ္တာနှင့် သမိုင်းဝင် မိသားစုဓာတ်ပုံ

၁၉၇၂ ခုနှစ်၊ ဧပြီလလောက်က ဖြစ်ပါတယ်။ လူသားတို့၏ အာကာသစူးစမ်းလေ့လာမှုသမိုင်းတွင် အရေးပါသော မှတ်တိုင်တစ်ခုဖြစ်သည့် Apollo 16 မစ်ရှင်ခရီးစဉ်အတွင်းမှာပေါ့။ ဒီခရီးစဉ်မှာ လိုက်ပါသွားခဲ့တဲ့ အာကာသသူရဲကောင်း Charles Duke ဟာ သိပ္ပံပညာဆိုင်ရာ စူးစမ်းမှုတွေအပြင် အလွန်စိတ်ထိခိုက်စရာကောင်းတဲ့၊ ကြည်နူးစရာကောင်းတဲ့ အမှတ်တရတစ်ခုကို လကမ္ဘာရဲ့ မျက်နှာပြင်ပေါ်မှာ ချန်ထားရစ်ခဲ့ပါတယ်။

လကမ္ဘာထက်မှာ ကျန်ရစ်ခဲ့တဲ့ ဖခင်မေတ္တာနှင့် သမိုင်းဝင် မိသားစုဓာတ်ပုံ

၁၉၇၂ ခုနှစ်၊ ဧပြီလလောက်က ဖြစ်ပါတယ်။ လူသားတို့၏ အာကာသစူးစမ်းလေ့လာမှုသမိုင်းတွင် အရေးပါသော မှတ်တိုင်တစ်ခုဖြစ်သည့် Apollo 16 မစ်ရှင်ခရီးစဉ်အတွင်းမှာပေါ့။ ဒီခရီးစဉ်မှာ လိုက်ပါသွားခဲ့တဲ့ အာကာသသူရဲကောင်း Charles Duke ဟာ သိပ္ပံပညာဆိုင်ရာ စူးစမ်းမှုတွေအပြင် အလွန်စိတ်ထိခိုက်စရာကောင်းတဲ့၊ ကြည်နူးစရာကောင်းတဲ့ အမှတ်တရတစ်ခုကို လကမ္ဘာရဲ့ မျက်နှာပြင်ပေါ်မှာ ချန်ထားရစ်ခဲ့ပါတယ်။ ဒီအကြောင်းအရာလေးကို မိတ်ဆွေတို့ ဖတ်ရှုရလွယ်ကူအောင် ကျွန်တော် ဆွေးနွေးတင်ပြပေးပါ့မယ်။

ခက်ခဲကြမ်းတမ်းသော ခရီးစဉ်နှင့် "လကမ္ဘာ အိုလံပစ်"

Apollo 16 ခရီးစဉ်ဟာ ချောမွေ့တဲ့ ခရီးစဉ်တော့ မဟုတ်ခဲ့ပါဘူး။ လကမ္ဘာပေါ်သို့ မဆင်းသက်မီမှာပင် Command Module ရဲ့ အင်ဂျင်ပိုင်းဆိုင်ရာ ချို့ယွင်းမှုတွေကြောင့် ဆင်းသက်ဖို့ အစီအစဉ်ကို ဖျက်သိမ်းရလုနီးပါး အခြေအနေမျိုး ကြုံတွေ့ခဲ့ရပါတယ် ။ ဒါပေမဲ့ ကံကောင်းထောက်မစွာနဲ့ပဲ ပြဿနာကို ဖြေရှင်းနိုင်ခဲ့ပြီး Charles Duke နဲ့ Commander John Young တို့ဟာ လကမ္ဘာပေါ်ကို ခြေချနိုင်ခဲ့ကြပါတယ်။

စိတ်ဝင်စားစရာကောင်းတာက ၁၉၇၂ ခုနှစ်ဟာ ကမ္ဘာမြေမှာ အိုလံပစ်ပွဲတော် ကျင်းပတဲ့နှစ်ဖြစ်တာကြောင့် သူတို့နှစ်ဦးဟာ လကမ္ဘာရဲ့ ဆွဲငင်အားနည်းပါးမှုကို စမ်းသပ်ရင်း "Moon Olympics" (လကမ္ဘာ အိုလံပစ်) ဆိုပြီး အပျော်သဘော ခုန်ပေါက်လှုပ်ရှားမှုတွေ ပြုလုပ်ခဲ့ကြပါသေးတယ်။ ဒီလိုလုပ်ဆောင်ရင်း John Young ဟာ နောက်ပြန်လဲကျသွားပြီး သူ့ရဲ့ ကျောပိုးအိတ် (Life Support System) နဲ့ ရိုက်မိခဲ့တာကြောင့် အသက်အန္တရာယ် စိုးရိမ်ရတဲ့ အခြေအနေနဲ့တောင် ရင်ဆိုင်ခဲ့ရပါသေးတယ် ။ အကယ်၍သာ အာကာသဝတ်စုံ ပေါက်ပြဲသွားခဲ့မယ်ဆိုရင် ချက်ချင်း အသက်ဆုံးရှုံးသွားနိုင်တဲ့ အခြေအနေမျိုးပါ။

လကမ္ဘာပေါ်က မိသားစု ပုံရိပ်

ဒီလို အသက်အန္တရာယ်ကြားကနေ Charles Duke ဟာ အလွန်နူးညံ့သိမ်မွေ့တဲ့ အလုပ်တစ်ခုကို လုပ်ဆောင်ခဲ့ပါတယ်။ သူဟာ လကမ္ဘာမျက်နှာပြင်ပေါ်မှာ ပလတ်စတစ်အိတ်နဲ့ သေချာထုပ်ပိုးထားတဲ့ မိသားစုဓာတ်ပုံလေးတစ်ပုံကို ချထားခဲ့ပါတယ် ။ အဲဒီဓာတ်ပုံရဲ့ ကျောဘက်မှာတော့ အခုလို စာရေးသားထားပါတယ်။

"ဒါကတော့ ၁၉၇၂ ခုနှစ်၊ ဧပြီလ၊ ၂၀ ရက်နေ့မှာ လကမ္ဘာပေါ်သို့ ဆင်းသက်ခဲ့သော ကမ္ဘာဂြိုဟ်မှ အာကာသယာဉ်မှူး Charlie Duke ၏ မိသားစု ဖြစ်ပါသည်။"

Charles Duke ဟာ ဒီဓာတ်ပုံအပြင် အမေရိကန် လေတပ်နှစ်ပတ်လည် အထိမ်းအမှတ်တံဆိပ်အချို့ကိုလည်း ထားရစ်ခဲ့ပါသေးတယ် ။

ဖခင်တစ်ယောက်ရဲ့ ကတိ

မိတ်ဆွေတို့ စဉ်းစားမိနိုင်တာက "ဘာကြောင့်များ ဒီဓာတ်ပုံကို သယ်သွားရတာလဲ" ဆိုတာပါပဲ။ Charles Duke က ၂၀၁၅ ခုနှစ် အင်တာဗျူးတစ်ခုမှာ ဖြေကြားခဲ့ဖူးပါတယ်။ သူ ဒီခရီးစဉ်မထွက်ခင်မှာ သူ့ရဲ့သားသမီးတွေကို စိတ်လှုပ်ရှားစေချင်တဲ့အတွက် "သားတို့သမီးတို့ အဖေနဲ့အတူ လကမ္ဘာကို လိုက်ချင်ကြလား၊ တို့မိသားစုဓာတ်ပုံကို ယူသွားရင် တို့တစ်မိသားစုလုံး လကမ္ဘာကို ရောက်သွားသလို ဖြစ်မှာပေါ့" လို့ ပြောခဲ့ကြောင်း ဆိုပါတယ် ။

သူဟာ ကလေးတွေကို ပြောခဲ့တဲ့အတိုင်း ဓာတ်ပုံကို လကမ္ဘာပေါ်မှာ တကယ်ပဲ ချန်ထားရစ်ခဲ့တာပါ။ "ကလေးတွေကို အဖေ တကယ်ပဲ လပေါ်မှာ ထားခဲ့တယ်ဆိုတာ ပြသချင်လို့ပါ" လို့လည်း Duke က ဆိုပါတယ် ။

နိဂုံးချုပ်

ယနေ့အချိန်ထိ ထိုဓာတ်ပုံလေးဟာ နေရောင်ခြည်ဒဏ်ကြောင့် အရောင်အဆင်းတွေ ပြောင်းလဲသွားနိုင်ပေမယ့် လကမ္ဘာရဲ့ ဖုန်မှုန့်တွေကြားမှာ တည်ရှိနေဆဲ ဖြစ်ပါတယ် ။ ဒါဟာ သိပ္ပံပညာရဲ့ အောင်မြင်မှု သက်သက်မဟုတ်ဘဲ၊ မိုင်ပေါင်း ၂ သိန်းကျော်ကွာဝေးတဲ့ အရပ်မှာတောင် မိသားစုကို သတိရနေတဲ့ ဖခင်တစ်ယောက်ရဲ့ မေတ္တာကို သက်သေပြနေတဲ့ သင်္ကေတတစ်ခုအဖြစ် ကျွန်တော် ရှုမြင်မိပါတယ်။
